# Calculator

## What it does
Takes two numbers and an operator 
```
+  addition
-  subtraction
*  multiplication
/  division
```
from the user and returns the result of the calculation.

## How it works
- User inputs first number
- User inputs an operator
- User inputs second number
- Program checks the operator and performs the correct calculation
- prints the results formatted to 2 decimal places: 10.00 / 2.00 = 5.00
